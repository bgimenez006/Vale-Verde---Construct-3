# 🌱 Vale Verde

Jogo 2D top-down desenvolvido no Construct 3, inspirado em mecânicas de jogos como Stardew Valley.

O projeto foca na implementação de sistemas de gerenciamento de recursos, economia e mecânicas baseadas em tempo, proporcionando uma experiência simples e interativa.

---

## 🎮 Funcionalidades

- 💰 Sistema de economia (controle de dinheiro do jogador)
- 🎒 Sistema de inventário com gerenciamento de itens
- 🌾 Mecânica de plantio com crescimento baseado em tempo
- 🧠 Lógica de interação com o ambiente

---

## 🛠️ Tecnologias utilizadas

- Construct 3  
- Lógica de programação  
- Estruturação de sistemas de jogo  

---

## 🚀 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de aplicar conceitos de lógica de programação e desenvolvimento de sistemas interativos, explorando mecânicas comuns em jogos de simulação.

---

## 📌 Aprendizados

Durante o desenvolvimento, foram trabalhados:

- Organização de lógica de sistemas  
- Gerenciamento de estados e variáveis  
- Desenvolvimento de mecânicas baseadas em tempo  
- Estruturação de funcionalidades interativas
